Justificación del Formulario

Nombre Usuario

Se requiere para identificar al usuario. Debe tener entre 3 y 16 caracteres y solo puede contener letras, números y guiones bajos.

Contraseña

Se requiere para la seguridad del usuario. Debe tener al menos 6 caracteres.

Repetir Contraseña

Se requiere para verificar que el usuario haya ingresado correctamente su contraseña.

Dirección

Se utiliza para la entrega de servicios o productos.

Ciudad

Se utiliza para identificar la ubicación del usuario. Debe contener al menos 2 caracteres y solo letras.

Provincia

Se utiliza para identificar la región del usuario. Debe contener al menos 2 caracteres y solo letras.

Número de Huéspedes

Se requiere para saber la cantidad de personas que se hospedarán.

Servicios brindados

Opción para seleccionar los diferentes servicios que nosotros le brindamos.

Tipo de habitación

Opción para seleccionar las diferentes habitaciones disponibles

Tipo de pensión

Opción para seleccionar el tipo de pensión

Comentarios

Opción para enviar su correo y dejar algún comentario o sugerencia

Contraseña

este campo de entrada está configurado para aceptar una contraseña que debe tener al menos 6 caracteres. Los caracteres ingresados se ocultan para mayor seguridad, el campo es obligatorio, y se proporciona un mensaje de ayuda para guiar al usuario.

Repetir Contraseña

este campo de entrada está configurado para aceptar una contraseña repetida (para confirmar la contraseña inicial) que debe tener al menos 6 caracteres. Los caracteres ingresados se ocultan para mayor seguridad, el campo es obligatorio, y se proporciona un mensaje de ayuda para guiar al usuario.

Dirección

Este campo de entrada permite al usuario ingresar su dirección completa. Muestra un texto de ejemplo cuando está vacío y proporciona un mensaje de ayuda cuando el usuario pasa el cursor sobre el campo.

Ciudad

Este campo de entrada permite al usuario ingresar el nombre de su ciudad, que debe contener al menos 2 caracteres y solo letras. Muestra un texto de ejemplo y proporciona un mensaje de ayuda si los datos no cumplen con el patrón.

Provincia

Este campo de entrada permite al usuario ingresar el nombre de su provincia, que debe contener al menos 2 caracteres y solo letras. Muestra un texto de ejemplo y proporciona un mensaje de ayuda si los datos no cumplen con el patrón.

Número de Huéspedes

Este campo de entrada permite al usuario ingresar el número de huéspedes, que debe estar entre 1 y 100. Muestra un texto de ejemplo y proporciona un mensaje de ayuda si los datos no están dentro del rango especificado.

Servicios brindados

Este campo de entrada permite al usuario seleccionar si desea aire acondicionado. En caso de estar seleccionado, se enviará el valor "Aire Acondicionado" con el formulario.

Tipo de habitación

Este campo de entrada permite al usuario seleccionar si desea una habitación individual. Solo se puede seleccionar una opción de este grupo y es obligatorio elegir una antes de enviar el formulario.

Tipo de pensión

Este campo de entrada permite al usuario seleccionar si desea solo alojamiento. Solo se puede seleccionar una opción de este grupo y es obligatorio elegir una antes de enviar el formulario.

Comentarios

-Este campo de entrada permite al usuario ingresar su dirección de correo electrónico, que debe ser válida según el patrón definido. Es un campo obligatorio y muestra un texto de ejemplo y proporciona un mensaje de ayuda si los datos no cumplen con el patrón.
-Opción para enviar su correo y dejar algún comentario o sugerencia. Usamos un textarea para permitir una entrada de texto más larga.
-Este campo de entrada permite al usuario aceptar la política de privacidad. Es un campo obligatorio para poder enviar el formulario.
